# GpsSyncTime

### CLI util for synchronize system time from GPS reciver

--------------------------------------
_Compatible with all GPS receivers that operate on the NMEA protocol_

Working on windows and linux.

----------------------------

## RELEASES:


---------------------

### config.yml caption

```yaml
port: /dev/ttyUSBS0
baud: 4800
```
